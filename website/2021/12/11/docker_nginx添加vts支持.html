docker nginx添加vts支持

2021-12-11 docker docker, nginx 0 Word Count: 882(words) Read Count: 5(minutes)

概览：

  • Dockerfile
  • 执行构建

docker hub上的nginx镜像默认不支持vts，故不支持prometheus监控。所以通过在官方的nginx的Dockerfile文件修改，添加vts支持。

先将pkg-oss，下载到本地，在其中的Makefile中的构建配置中添加--add-module=/data/src/nginx-module-vts。

然后根据实际情况将Dockerfile里面的拉取nginx-module-vts和pkg-oss换成你的地址。就可以执行docker构建。
